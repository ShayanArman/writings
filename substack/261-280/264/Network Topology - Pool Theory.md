Most modern networks that we can think about (Twitter/X, LinkedIn, Instagram) are run top-down, with a few two-node networks here and there. The problem with this Topology is that each of the edges between nodes (People in the Network) are fairly weak. Why? The Nodes at the Top have no way of possibly communicating / managing relationships with all those people. Healthier Networks are run in both directions, bottom-up (think institutions at the GrassRoots level) + top-down (think Federal Reserve). My theory on Networks is a focus on Pools, where we create connections in a more genuine manner. Inflection @ Waterloo is a focused Alumni+Student Network We're doing pretty heavy vetting / chats.
